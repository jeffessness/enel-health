Build source.

The complete static-site generator behind the landing funnel. One Node script produces all 27 study pages, the 13 gated library pages, and the hub index. No backend; the signup form is a demo.

Download

Download everything (.zip) build_rework.js README

The zip contains the generator, the image assets (hero photos and the 13 rendered library screenshots), the local preview server, and the README.

Files

build_rework.jsThe generator. All copy, palettes, and structure live in the STUDIES, PALETTES, and LIBRARIES objects.
serve.jsTiny static server for local preview on port 5599.
README.txtBuild steps and notes.
assets/Hero photos and the 13 library screenshots (libshot-*.png).

Build

node build_rework.js out
cp -r assets out/assets        # windows: xcopy /E /I assets out\assets
node serve.js                  # then open http://localhost:5599

Edit the copy or structure in build_rework.js and re-run. The output is self-contained static HTML.
