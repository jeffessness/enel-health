The pages for each study

You will see three files per study, named simply:

  • ..._landing.html — the intro page that explains the study.
  • ..._join.html — the sign-up page.
  • ..._payoff.html — the thank-you page after someone signs up.

So, for example, fasting_join.html is the sign-up page for the Fasting study.

To open: double-click index.html to preview them together, or open any single page. If one looks blank, right-click and Open with → Chrome.
